# Smart Attendance Monitoring Through Facial Recognition

## Overview
This project is an AI-powered attendance monitoring system that uses facial recognition technology to automatically detect and mark student attendance.

## Features
- Real-time face detection
- Facial recognition-based attendance
- Flask web application
- Webcam integration
- Automated attendance records

## Technologies Used
- Python
- Flask
- OpenCV
- HTML
- CSS
- JavaScript

## Project Modules
1. Student Face Registration
2. Face Training Module
3. Attendance Marking System
4. Attendance Record Management

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the Flask application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
